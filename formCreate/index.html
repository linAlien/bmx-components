# API

# Props

参数 说明 类型 默认值 可选值
imageUploadHandler 图片上传方法(需返回promise) Function - -

# Methods

此方法如要通过ref手动调用

名称 说明 参数 返回值
init 传递生成表单数据进行初始化 { data: [], model: ''} -
setGlobalDisabled 设置禁用 true|false -
submit 提交校验 - promise
getData 获取答案 - Array
