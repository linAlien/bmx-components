# Hbuilder X方式

使用Hbuilder X打开工程，在uni-app插件市场通过uni_modules的形式进行安装，此安装方式可以方便您后续在uni_modules对uView进行一键升级。

# 使用方法

引用内部修改过的uView，引用css文件

//main.js
import uView from '@/uni_modules/bmx-components/uview-ui';
Vue.use(uView);

//uni.scss
@import "@/uni_modules/bmx-components/uview-ui/theme.scss";
@import "@/uni_modules/bmx-components/theme.scss";

// 覆盖主题色
$bmx-theme: #3379ff;
$bmx-theme-background: #ecf1ff;

组件需使用vuex

// store
import Vue from 'vue';
import Vuex from 'vuex';
import formCreate from '@/uni_modules/bmx-components/libs/store/modules/form-create.js';


Vue.use(Vuex);

const store = new Vuex.Store({
  modules: {
    formCreate,
  },
});

export default store;

//main.js
import store from './store';
Vue.prototype.$store = store;

组件需配置easycom规则后，自动按需引入，无需import组件，直接引用即可，由于业务需求插件中包含已修改过uView组件也需配置easycom规则引用。uView组件可正常使用

"easycom": {
	    "autoscan": true,
	    "custom": {
	      "^u-(.*)": "@/uni_modules/bmx-components/uview-ui/components/u-$1/u-$1.vue",
		  "^bmx-(.*)": "@/uni_modules/bmx-components/components/bmx-$1/bmx-$1.vue"
	    }
}
